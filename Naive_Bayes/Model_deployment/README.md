# 📧 SMS/Email Spam Detector

A machine learning web app that detects whether a message is Spam or Not Spam using **Naive Bayes** algorithm.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://traditional-ml-algorithms-alnm5q2v3e8ezmcci8evuv.streamlit.app/)

## 🚀 Live Demo
👉 **[Click here to try the app!](https://traditional-ml-algorithms-alnm5q2v3e8ezmcci8evuv.streamlit.app/)**

## 📌 How it works
- Enter any SMS or Email message
- Click **Predict**
- App tells you if it's **Spam** or **Safe**

## 🛠️ Built With
- Python
- Streamlit
- Scikit-learn
- NLTK
- Naive Bayes Algorithm
